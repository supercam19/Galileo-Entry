Data for this website comes from NASA's Planetary Data System https://pds-atmospheres.nmsu.edu/data_and_services/atmospheres_data/Galileo/galileo.html.
Entry and descent data are from the ASI instrument, atmospheric composition is from NMS.

Raw data files are in the /raw directory.
ratios.tab has been simplified for the purposes of this dashboard, find the original at the above link.

accel.txt describes entry acceleration data. descent.lbl gives column definitions for raw/descent.tab. ratios.lbl gives column definitions for raw/ratios.tab.
